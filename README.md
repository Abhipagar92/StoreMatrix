# StoreMatrix - Store Rating & Management Platform

## 📌 Project Overview

StoreMatrix is a full-stack web application that enables users to discover stores, submit ratings, and view store information. Store Owners can monitor ratings and store performance, while Administrators can manage users, stores, and store owners through a centralized dashboard.

The application follows a Role-Based Access Control (RBAC) architecture with three user roles:

* Admin
* Store Owner
* Normal User

The project is fully responsive and deployed using modern cloud platforms.
